# BDEVidencija PWA v3.0.0

**Vaša digitalna evidencija vožnji i dostava - B&D Kleintransporte KG**

Progressive Web App verzija naše Android aplikacije sa modernizovanim UI/UX dizajnom.

## 🚀 Verzija 3.0.0 - PWA

Kompletna migracija sa Android native aplikacije na Progressive Web App sa:
- ✨ Glassmorphism UI efekti  
- 📱 Cross-platform podrška
- ⚡ Poboljšane performanse
- 🎨 Responsive design
- 🌐 Web aplikacija dostupna na svim uređajima

## 🛠️ Tech Stack

- ⚛️ **React 19** - Frontend framework
- ⚡ **Vite** - Build tool i dev server  
- ☁️ **Supabase** - Backend-as-a-Service
- 🎨 **CSS3** - Glassmorphism styling
- 📱 **PWA** - Progressive Web App podrška

## 🚦 Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
